About Me

Hi, I'm Mbogo Eston Kimani — a passionate, self-taught full-stack developer, UI/UX designer, and creative problem solver. With a strong foundation in software programming and a track record of building dynamic, high-performance web applications, I specialize in crafting clean, responsive, and engaging digital experiences.

I’ve developed a wide range of modern web solutions — from full-featured dashboards, eCommerce backends, and SaaS platforms to branding-rich landing pages and interactive user interfaces. My stack includes Vue.js, Node.js, Express, MongoDB, REST APIs, Chart.js, and more. I also bring strong front-end design skills using tools like Figma and Adobe XD, seamlessly blending form and function.

Beyond coding, I’ve actively contributed to hackathons and tech spaces, collaborating with fellow developers and designers to solve real-world challenges under tight deadlines. These experiences have strengthened my ability to think critically, work in teams, and deliver quality under pressure.

I'm also the founder of Evo Tech Ltd., a growing digital agency focused on delivering innovative web solutions, digital branding, and SEO services. Through this venture, I’ve deepened my expertise in business tech strategy, API integrations, scalable architecture, and client-focused development.

I’m continuously improving — exploring automation, M-Pesa/other payment methods integrations, performance optimization, AI Integration and advanced UI/UX techniques. I believe in learning by doing, and I’ve proven it by building, deploying, and scaling real products — including eCommerce platforms and digital dashboards hosted on platforms like Vercel and Render.

My mission is to build impactful, intuitive digital products that not only solve problems but also leave lasting impressions. I'm driven by curiosity, creativity, and the desire to lead in a tech-forward future.

Development Tools

  • Html used for structuring websites, providing the foundational layout and organization of content on web pages.
  • Css utilized for styling and animating websites, allowing for the customization of visual elements such as colors, fonts, layouts, and animations.
  • JavaScript employed for adding interactivity, animation, and responsiveness to websites. It enables dynamic behavior, form validation, and the manipulation of HTML and CSS.
  • Node.js is a JavaScript runtime built on Chrome’s V8 engine that enables fast and scalable backend development. It streamlines the creation of high-performance server-side applications using a non-blocking, event-driven architecture.
  • Express.JS is a minimal and flexible Node.js web application framework that simplifies server-side development. It provides a robust set of features for handling routing, middleware, and HTTP requests, making backend APIs efficient and maintainable.
  • Vue.Js is a progressive JavaScript framework for building dynamic user interfaces and single-page applications. It offers a flexible and component-based architecture that enhances development speed while maintaining clean and modular code.
  • React.Js is a powerful JavaScript library commonly used for building complex and interactive user interfaces. It excels in creating large-scale, single-page applications with reusable components.
  • Tailwind CSS is a utility-first CSS framework that facilitates rapid development by providing pre-defined classes for styling and layout. It streamlines the process of creating responsive and visually appealing designs for large-scale websites.

Storage/Databases

  • Google Clouds is a suite of cloud computing services that supports scalable, secure, and highly available infrastructure. It accelerates the deployment of applications and services through tools for compute, storage, databases, and machine learning.
  • Mysql is a reliable, open-source relational database management system ideal for structured data storage. It supports complex queries, transactions, and indexing, making it well-suited for data-intensive and enterprise-level applications.
  • MongoDb is a flexible, document-oriented NoSQL database designed for scalability and high availability. It simplifies handling unstructured or evolving data by storing information in JSON-like documents with dynamic schemas.

Design Tools

  • Figma is a collaborative interface design tool used for creating interactive prototypes, wireframes, and user interface designs for websites and applications. Its cloud-based platform enables real-time collaboration among team members, making it ideal for designing and iterating on complex projects.
  • Adobe Illustrator is a versatile graphic design software widely used for creating and editing images, illustrations, and graphics. It offers advanced features for photo manipulation, digital painting, and graphic design, making it suitable for designing logos, editing photos, and creating mockups for websites and applications.
  • Canva is a user-friendly online graphic design platform that offers a wide range of tools and templates for creating professional-looking graphics, including logos, social media posts, presentations, and videos. It simplifies the design process with its drag-and-drop interface and extensive library of customizable templates, fonts, and graphics.
